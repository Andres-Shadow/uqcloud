Máquinas virtuales de corta duración
Estas máquinas virtuales al ser de corta duración ofrecen un entorno limpio o específico según el tipo de instacia, no se mantienen persistentes, lo que significa que cualquier cambio o información almacenada en ellas se perderá una vez que la máquina física se apague (interrumpa).
Máquinas virtuales CLI
Para este tipo de máquinas virtuales se accede y controla a través de una interfaz de línea de comandos, la gestión del sistema operativo se realiza de forma directa, no poseen interfaz gráfica de usuario (GUI), ya que su caso de uso se centra en la creación desde otros dispositivos.
Máquinas virtuales GUI
Estas máquinas solo se generan al solicitar su creación desde un dispositivo que actúa como host dentro de la plataforma. Este proceso permite a los usuarios interactuar directamente con la máquina virtual sin necesidad de utilizar otro dispositivo, con la finalidad de que la interfaz gráfica no interfiera a otros usuarios de diferentes dispositivos.
Usuarios invitados
Los usuarios invitados pueden acceder al sistema sin necesidad de registrarse, lo que les permite usar la plataforma y crear máquinas virtuales de manera temporal. Al no tener credenciales, no se manteinen persistentes.
